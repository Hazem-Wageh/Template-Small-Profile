# Template-Small-Profile

## 📌 Overview
**Template-Small-Profile** is a modern and responsive personal profile template built using **HTML5** and **CSS3**.  
This template is perfect for showcasing portfolios, personal profiles, or small web projects with a clean and professional design.

---

## 🚀 Features
- ✅ Modern and clean UI  
- ✅ Fully responsive across all devices (mobile, tablet, desktop)  
- ✅ Well-structured, readable, and maintainable code  
- ✅ Smooth layout with media queries  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3 (Flexbox, Grid, Media Queries)
